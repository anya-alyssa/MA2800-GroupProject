4. What this website contains (summary):
   This website contains our Live Report projects, which capture three very different real-time events. The first project page showcases each group member’s 2-5 minute live report, alongside key details such as title, location, duration, and date. The research page provides a record of our individual processes, including individual summaries, facts on our events, planning, research inspirations, and more.
